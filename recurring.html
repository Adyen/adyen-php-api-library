Recurring

The payment service has the following methods:

listRecurringDetails

Using you can retrieve all cards from a specific shopper. 

$service = new \Adyen\Service\Recurring($client);

$recurring = array('contract' => \Adyen\Contract::RECURRING);

$params = array(
    "merchantAccount" => 'YOUR MERCHANT ACCOUNT',
    "recurring" => $recurring,
    "shopperReference" => 'YOUR SHOPPER REFERENCE'
);

$result = $service->listRecurringDetails($params);

disable

$service = new \Adyen\Service\Recurring($client);

$params = array(
    "merchantAccount" => 'YOUR MERCHANT ACCOUNT',
    "shopperReference" => 'SHOPPER_REFERENCE',
    "recurringDetailReference" => 'RECURRING_DETAIL_REFERENCE'
);

$result = $service->disable($params);
