Modification

The payment service has the following methods:

capture

Using this you can capture the payment. 

$service = new \Adyen\Service\Modification($client);

$modificationAmount = array('currency' => 'EUR', 'value' => '750');

$params = array(
    "merchantAccount" => 'YOUR MERCHANT ACCOUNT',
    "modificationAmount" => $modificationAmount,
    "originalReference" => 'ORIGINAL PSP REFERENCE OF AUTHORISED PAYMENT'
);

$result = $service->capture($params);
// result['response'] should be [capture-received]

cancel

$service = new \Adyen\Service\Modification($client);

$params = array(
    "merchantAccount" => 'YOUR MERCHANT ACCOUNT',
    "originalReference" => 'ORIGINAL PSP REFERENCE OF AUTHORISED PAYMENT'
);

$result = $service->cancel($params);
// $result['response'] should be [cancel-received]

cancelOrRefund

Similar to cancel, using this you can choose to cancel or refund the payment.

$service = new \Adyen\Service\Modification($client);

$params = array(
    "merchantAccount" => 'YOUR MERCHANT ACCOUNT',
    "originalReference" => 'ORIGINAL PSP REFERENCE OF AUTHORISED PAYMENT'
);

$result = $service->cancelOrRefund($params);
// $result['response'] should be [cancelOrRefund-received]

refund

With refund you have to specify the amount you want to refund. You can also choose to do a partial refund. 

$service = new \Adyen\Service\Modification($client);

$modificationAmount = array('currency' => 'EUR', 'value' => '750');

$params = array(
    "merchantAccount" => 'YOUR MERCHANT ACCOUNT',
    "modificationAmount" => $modificationAmount,
    "originalReference" => 'ORIGINAL PSP REFERENCE OF AUTHORISED PAYMENT'
);

$result = $service->refund($params);
// result['response'] should be [refund-received]
